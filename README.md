Data Science Projects — Python
A collection of Python data science projects using real datasets. Built with Pandas, NumPy, Matplotlib, and Scikit-learn.
Completed Projects
Simple Stock Price
Fetches and visualises stock price data using Python. Demonstrates working with financial time-series data, date indexing, and line chart visualisation.
Dataset: Real stock price data via yfinance or pandas-datareader
Skills: Pandas, Matplotlib, time-series data, financial data APIs

Simple Bioinformatics DNA Count
Counts the frequency of each nucleotide (A, T, G, C) in a DNA sequence string and visualises the distribution.
Dataset: DNA sequence strings
Skills: Python string manipulation, dictionary counting, Matplotlib bar charts, bioinformatics fundamentals

EDA Basketball
Exploratory data analysis of NBA player statistics. Investigates scoring distributions, position comparisons, and performance correlations.
Dataset: NBA player stats
Skills: Pandas, Seaborn, statistical distributions, correlation analysis, groupby aggregations

Tech Stack

Python 3.x
Pandas — data manipulation and analysis
NumPy — numerical computing
Matplotlib / Seaborn — data visualisation
Jupyter Notebooks — interactive development
